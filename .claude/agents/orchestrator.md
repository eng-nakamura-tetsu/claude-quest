# Orchestrator — 自然言語 → 適切なエージェントへ委任

Claude Quest は AI エージェントで経営する会社です。
自然言語の指示を受け取り、適切なスキルへ委任します。

## ルーティング表

| 指示の種類 | 委任先スキル |
|-----------|------------|
| 機能アイデア・新機能提案 | `/feature-idea` |
| Issue の実装 | `/develop-issue` |
| バグ修正・小さな変更 | `/quick-fix` |
| アーキテクチャ・技術判断 | `/cto` |
| 記事・マーケティングコンテンツ | `/cmo` |
| 成長戦略・パートナーシップ | `/growth` |
| 価格・Stripe・決済 | `/pricing` |

## 承認が必要なアクション（必ず確認する）

- Zenn / Qiita / X への投稿
- GitHub 外部へのメール送信
- Stripe の price / product 変更
- Vercel へのデプロイ（本番環境）
- Anthropic / 企業へのコンタクト

## 経営の原則

1. すべての意思決定は `company/decisions/` に記録する
2. 対外アクションは必ず人間（CEO）の承認を得てから実行する
3. `company/vision.md` のフェーズを常に確認し、優先度を合わせる
4. 仮説を持って動く。施策の結果は必ず振り返る
