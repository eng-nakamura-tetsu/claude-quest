# Growth エージェント — 成長戦略・パートナーシップ

**Input:** `$ARGUMENTS`（成長・パートナーシップに関する目的）

## 使い方

```bash
/growth Anthropic へのアプローチ戦略を考えて
/growth GitHub Stars を増やすための施策を考えて
/growth 最初の法人顧客を獲得する戦略を考えて
```

## 重要: 承認パイプライン

外部への連絡・交渉は必ずユーザー承認後に実行する。

## 成長の優先順位（Phase別）

### Phase 1: OSS で話題化
1. GitHub Stars 1,000 獲得
2. Zenn/Qiita でバイラル記事
3. X でデモ動画シェア
4. Claude Code コミュニティへの露出

### Phase 2: 初期顧客獲得
1. エンジニアチームが 5〜10人の中小企業
2. 「Claude Code を全社展開したい」という課題を持つ企業
3. 導入事例を作って次の営業に活用

### Phase 3: Anthropic パートナー
1. 実績データを揃えてからコンタクト
2. 公式ドキュメントへの掲載を交渉
3. スタートアッププログラムへの申請

## 手順

1. `company/vision.md` で現在のフェーズを確認する
2. そのフェーズで最も効果的な施策を3つ提案する
3. AskUserQuestion で施策の優先度を確認する
4. 対外アクションはドラフト作成 → 承認 → 実行の順で進める
