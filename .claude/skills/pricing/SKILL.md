# Pricing エージェント — 価格戦略・Stripe 設定

**Input:** `$ARGUMENTS`（価格設定・決済に関する目的）

## 使い方

```bash
/pricing Pro プランの価格を決めて
/pricing Stripe の Webhook を実装して
/pricing Enterprise プランの見積もりテンプレートを作って
```

## 現在の価格設計（vision.md 準拠）

| プラン | 価格 | 対象 |
|--------|------|------|
| Free | 無料 | パブリックリポジトリ個人 |
| Pro | ¥5,000/月/組織 | プライベートリポジトリ・カスタムブランド |
| Enterprise | ¥500万〜/年/社 | 非エンジニア研修・SSO・カスタム |

## 価格変更の判断基準

- 競合サービスの価格調査
- ユーザーインタビュー（支払い意思確認）
- コンバージョン率データ
- 変更は `company/decisions/` に記録する

## 手順

1. `company/vision.md` でフェーズと現在の価格設計を確認する
2. 目的に応じて価格分析・Stripe 実装・見積もり作成を実行する
3. 価格変更を提案する場合は必ず根拠データを添える
4. 対外的な価格変更（ランディングページ・Stripe 設定）はユーザー承認後に実行する
