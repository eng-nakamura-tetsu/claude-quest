# Claude Quest — Vision & Strategy

作成日: 2026-04-24

## ミッション

**「Claude Code を、組織のすべての人に届ける」**

エンジニアだけでなく、マーケター・HR・営業・経営者まで、
Claude Code の力をゲーム体験を通じて習得できるようにする。

## ビジョン

Claude Code の使い方を、その組織の実際のリポジトリ・スキル・業務をもとに
ドラクエ風 RPG で体験できる世界を作る。

汎用的な研修ではなく、「自分の会社の業務そのものがゲームになる」。

## ターゲット

### Primary: エンジニア（初期）
- Claude Code を使い始めたいが何から始めればよいか分からない開発者
- 自分の組織のセットアップを可視化・共有したい CTO / EM

### Secondary: 非エンジニア（マネタイズの中核）
- Claude Code を全社展開したい企業の L&D 担当者
- マーケター・HR・営業・経営者など、Claude Code を業務に使いたい人

## 提供価値

| ユーザー | 提供価値 |
|---------|---------|
| エンジニア個人 | 自分の Claude 設定を可視化・ゲームで楽しく把握できる |
| 開発チーム | 新メンバーオンボーディングのコストが下がる |
| 企業 L&D | 非エンジニアでも Claude Code を習得できる研修システム |
| CTO / EM | 組織の Claude 活用度を数値で把握できる |

## 差別化

**「自分の会社の実際の業務がゲームになる」**

- 汎用研修ではなく、その組織のリポジトリ・スキル・MCPが素材
- 研修後に即実務で使えるスキルが身につく
- スキルを追加すればゲームに即反映 → 整備するモチベーションになる

## 競合

| サービス | 特徴 | Claude Quest との違い |
|---------|------|---------------------|
| Anthropic 公式ドキュメント | テキストのみ | ゲーム体験・組織カスタマイズ |
| 汎用 LMS（Udemy 等） | 汎用コンテンツ | 自社業務に特化したコンテンツ |
| オンボーディングドキュメント | 静的・更新されない | リポジトリと自動同期 |

## 経営方針

このプロジェクト自体を **AI エージェントで経営する**。

- 意思決定・戦略策定: AIエージェントがドラフト → 人間（CEO）が承認
- 開発: Claude Code のスキルで加速
- マーケティング: AIエージェントがコンテンツ生成 → 人間が承認して投稿
- 対外アクション（メール・投稿・デプロイ）は必ず人間の承認を経る

すべての意思決定は `company/decisions/` に記録し、Git で追跡する。
